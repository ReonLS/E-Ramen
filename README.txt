Project Name : E-Ramen
Designs: Figma, Canva
Language: HTML, CSS, JavaScript

E-ramen is an online website designed for online user to collect information around E-Ramen. The website allows user to sign up to get newsletter around E-ramen as well as see the dishes served in E-Ramen. User can also learn the history of E-Ramen, learning how E-Ramen came to be founded as well as the goals of E-Ramen
